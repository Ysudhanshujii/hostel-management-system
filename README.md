# Hostel Management System

A full-stack Hostel Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## Features

### Student Module

* Student Registration
* Room Availability
* Room Allocation View
* Room Change Request
* Complaint Management
* Mess Weekly Menu
* Food Feedback & Rating

### Admin Module

* Admin Login
* Student Management
* Room Allocation
* Room Change Request Approval
* Complaint Monitoring
* Mess Menu Management
* Feedback Monitoring
* Dashboard Analytics

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB
* Version Control: Git & GitHub

## Installation

```bash
npm install
```

```bash
npm start
```
